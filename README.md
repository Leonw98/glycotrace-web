## Features
- **Invisible Sync**: Automatic background sync via LibreLinkUp.
- **Enzyme Tracker**: Specialized tracking for pancreatic insufficiency.
- **Garmin Photo Import**: OCR-based exercise logging from photos.
- **Fasting Intelligence**: Fat-burning forecast based on glucose stability.
- **Rapid Calorie Tracking**: Optimized food logging interface.
- **Glucoflow AI**: Historical data analysis assistant.
